Contribute to AVN Innovations

We welcome learners, developers, and researchers to join us in building ethical and impactful tools in Artificial Intelligence and Cybersecurity.

Ways You Can Contribute

  • Suggest new ideas or improvements
  • Contribute tools, code, or integrations
  • Improve tutorials and documentation
  • Help test features or report bugs
  • Join AI or Cybersecurity projects
  • Receive mentorship if you're just starting out

New to GitHub?

We recommend creating a free GitHub account to fork, edit, and contribute with ease. It helps track your contributions and build your open-source profile.

Don't Have a GitHub Account?

No problem — you can still become a member of AVN Innovations and contribute in your own way!

  • 🔒 Tools, code, and research are hosted on GitHub
  • 📁 You won’t be able to access project files without GitHub
  • 📥 Still contribute ideas, blog content, research topics, and more
  • 🧑‍🤝‍🧑 Join events, discussions, and collaborations
  • 💬 We can guide you through GitHub setup

🚀 You Don’t Need Advanced Coding Skills

All we ask for is curiosity, collaboration, and a willingness to learn. Many members started as beginners and became powerful contributors.

📘 Read Our Contribution Guide

📜 Code of Conduct

We are committed to a respectful, inclusive, and professional space. Our values include:

  • Respect for all backgrounds and skill levels
  • Support for beginners and ethical learning
  • Constructive, professional communication
  • Ethical innovation in AI and Cybersecurity
  • Clear protection of intellectual work

🚫 No harassment, plagiarism, spam, or unethical behavior allowed.

💡 Want to directly contribute your work? You can upload it here!

Upload any file you'd like to contribute -this could be:

Code

Research or blog drafts

UI designs

Testing Reports or bug ideas

Any creative ideas you want to share

🔹 📁 Before You Upload Your File

> Please read the following file guidelines carefully before uploading your contribution:

✅ Allowed File Types:

For written content (blogs, research, documentation) - .pdf, .docx, .txt

For code, scripts, and zipped project folders - .zip, .py, .ipynb

For designs, diagrams, UI/UX contributions - .png, .jpg, .fig

⏳ File Size Limit:

Max file size: 10 MB per file

You can upload up to 3 files (depending on your form settings)

📌 Make Sure:

Your file is original or clearly credited if using others' work

It’s clearly named (e.g., phishing_blog_draft.docx or ai_model_code.zip)

You’ve described the file.
